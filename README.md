# SGCraft
## Français

![alt text](src/main/resources/images/Stargate_Minecraft.png "Stargate banniere")


### Pourquoi ce projet ?
Ce projet à pour objectif de mélanger ma passion de Stargate et mon envie de réaliser un mod Minecraft me permettant une grande liberté de les développements possibles.

### Projet d'origine
Ce projet est en grande partie basé sur le travail de Greg Ewing. Tout le mérite de ce mode lui revient.
Le projet d'origine est disponible à cette adresse : https://www.cosc.canterbury.ac.nz/greg.ewing/minecraft/mods/SGCraft/
